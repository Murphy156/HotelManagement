# 介绍
酒店（客房）管理系统

## 主要功能特性
### 信息录入：
- 人：租客信息录入
- 物：房屋信息录入
- 关系：每月收入流水录入

### 数据统计分析
- 大盘统计
- 门店分析
- 房务KPI
